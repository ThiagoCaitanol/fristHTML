Um pouco de Historia

Um pouco de Historia

O HTML, sigla em inglês para Hypertext Markup Language, é a linguagem de marcação padrão para a criação de páginas web. Ele foi criado por Tim Berners-Lee, em 1991, enquanto trabalhava no CERN, na Suíça. A primeira versão do HTML foi bastante simples, com poucos elementos disponíveis para a formatação de texto e imagens. Com o tempo, novas versões do HTML foram lançadas, introduzindo novos recursos, como a capacidade de criar tabelas, formulários e animações. Em 2014, foi lançado o HTML5, que trouxe ainda mais recursos, como a possibilidade de incluir vídeos e áudio diretamente nas páginas web. Atualmente, o HTML é a base para a criação de quase todas as páginas da web e continua evoluindo para atender às necessidades dos desenvolvedores e usuários.

Diferenças Entre Cliente e Servidor

Cliente e servidor são duas entidades diferentes em um modelo de comunicação em rede. O cliente é um dispositivo que solicita um serviço ou recurso a um servidor, enquanto o servidor é o dispositivo que fornece o serviço ou recurso solicitado pelo cliente. O cliente é tipicamente um computador pessoal, tablet ou smartphone, enquanto o servidor é geralmente um computador poderoso com recursos de processamento e armazenamento em rede. A comunicação entre cliente e servidor é estabelecida através de um protocolo, como HTTP ou FTP, que permite que os dados sejam transmitidos entre as duas entidades. Em resumo, o cliente faz a solicitação e o servidor atende a solicitação, tornando a comunicação possível e eficiente.

Linguagens de Programação

Linguagens de programação são conjuntos de instruções usadas para desenvolver softwares e aplicativos. Elas são compostas por uma série de comandos que indicam ao computador o que fazer. As três linguagens de programação mais utilizadas atualmente são:

  1. Java: uma linguagem de programação de alto nível, que é usada para desenvolver aplicativos móveis, sistemas web e jogos.
  2. Python: uma linguagem de programação de alto nível, que é usada para desenvolver aplicativos, análise de dados e inteligência artificial.
  3. JavaScript: uma linguagem de programação interpretada, que é usada para desenvolver aplicativos web, jogos e aplicativos móveis.

Tags e seus Atributos

Principais tags:

  • <html> define o início e o fim do documento HTML.
  • <head>: contém informações sobre o documento, como o título, as meta tags e os links para arquivos CSS e JavaScript.
  • <title>: define o título da página que é exibido na barra de título do navegador.
  • <body>: contém o conteúdo principal do documento, como texto, imagens, formulários, etc.
  • <h1> - <h6>: define os seis níveis de títulos do documento, sendo h1 o título principal e h6 o título de menor nível.
  • <p>: define um parágrafo de texto.
  • <a>: cria um link para outra página ou recurso.
  • <ul> e <li>: cria uma lista não ordenada, onde cada item é marcado com um ponto.
